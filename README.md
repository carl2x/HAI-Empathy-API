# HAI-Empathy
Customized API to talk to Furhat.

## Requirements
```
python >= 3.11
```

## Getting started
```bash
git clone https://github.com/carl2x/HAI-Empathy-API
cd HAI-Empathy-API
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python src/dummy.py
```
